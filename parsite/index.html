Welcome to the Parsite Docs!

Parsite is a small library containing parser combinators for parsing strings.

The library provides functions and types for a few string operations that may prove useful. Provided in monadic shape!

  • Installation
  • Usage
  • Get Started!

Installation

To install this library, update your available packages in opam with opam update, and then run opam install parsite.

Usage

Parsite has 2 modules: Combinators and Types. To include the modules in your file, add them to your dune project accordingly, and do open Parsite.Combinators or open Parsite.Types according to your needs.
